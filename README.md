# 🚆 Railway Ticket Reservation System

A menu-driven Railway Ticket Reservation System developed in **C Programming** as a mini project. The application allows users to book, search, update, cancel, save, and load railway ticket reservations using file handling.

---

## 📌 Features

- 🎫 Book Railway Tickets
- 🔍 Search Ticket by Ticket Number
- 📋 Display All Booked Tickets
- ✏️ Update Ticket Details
- ❌ Cancel Ticket
- 💾 Save Ticket Data to File
- 📂 Load Ticket Data from File
- 🚉 Predefined Train Information
- 🪑 Date-wise Seat Availability
- ✅ Input Validation
- 🖥️ User-friendly Console Interface

## 🚀 How to Run

### Compile

```bash
gcc main.c -o railway
```

### Run

**Windows**

```bash
.\railway.exe
```

**Linux/macOS**

```bash
./railway
```
